# BDI Company

BDI Company is a freelance project developed for a construction company based in the United States.

The goal of the project was to design and develop a modern, professional and responsive business website that showcases the company's construction services and completed projects.

## Technologies

- Astro
- HTML
- CSS
- JavaScript

## Features

- Fully responsive design.
- Modern and clean user interface.
- Home page with company presentation.
- Services section.
- About Us section.
- Contact page.
- Embedded Google Maps location.
- Image galleries.
- Construction project videos.
- Contact form using FormSubmit.
- SEO-friendly structure.
- Optimized performance.

## My Responsibilities

- Website design implementation.
- Frontend development.
- Responsive layout.
- Integration of Google Maps.
- Contact form configuration.
- Deployment using Vercel.
- Domain configuration.
- Performance optimization.

## Project Goal

The objective was to create a professional online presence for the client, allowing potential customers to explore previous work, learn about the company's services and request a free estimate through the website.

The website was successfully delivered to the client and is currently available online.

# TrasteRush

TrasteRush is Rafael García's Final Degree Project for the Higher National Diploma in Multiplatform Application Development (DAM).

It is a full-stack web application designed to manage the rental of storage units.

## Technologies

Frontend:
- Angular
- TypeScript
- HTML
- CSS

Backend:
- PHP
- REST API

Database:
- MySQL / MariaDB

Deployment:
- Google Cloud Platform (GCP)

## Features

- User registration and login.
- JWT authentication.
- Protected routes.
- User profile management.
- Storage unit management.
- Rental management.
- Administrator dashboard.
- User administration.
- Responsive interface.
- CRUD operations.
- REST API communication.

## Administrator Panel

The administrator has access to a protected dashboard where they can:

- Manage users.
- Manage storage units.
- Create, edit and delete information.
- Monitor the status of storage rentals.

## Project Goal

The objective of TrasteRush was to build a complete full-stack application applying best practices in frontend and backend development.

The project was successfully presented as Rafael's Final Degree Project and received positive feedback from the evaluation panel.
